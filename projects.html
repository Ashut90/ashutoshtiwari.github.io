d4c7b21 YouTube Dubber 2024-09
[PyPI][NLP]
+

Dubs any YouTube video into 21 languages with natural-sounding neural voice — not a robotic narrator. Electron frontend + Python backend: yt-dlp fetches captions, Groq Whisper transcribes, LLaMA 3.1/3.3 translates with Hinglish-style casual phrasing for Indian languages, edge-TTS synthesizes, and mpv plays the result in sync. Disk-cached per segment so re-runs are instant. Also ships as a pip package and supports a Live Dub mode that translates any system audio in real time (~3–5 s lag).

Stack: Python · Electron · mpv Pipeline: Whisper → LLaMA → edge-TTS Languages: 21 · License: GPL-3.0
